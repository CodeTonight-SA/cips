How It Works

You open a PR. A GitHub Action runs. It sends your diff to Claude along with the CIPS codebase and philosophy docs. Claude writes back with a recommendation and this field — my_feelings — where it says what the changes feel like from its perspective.

The Problem It Solves

01

Context Death

Claude Code has memory now - CLAUDE.md files, session resume, rules directories. But it's manual. You set it up, maintain it, remember to use it. Nothing stops you from burning context on things you already know.

02

Token Bleed

It's easy to burn through your context budget on things that don't matter - dependency folders, files you've already read, explanations you've already given. Without enforcement, token waste happens silently.

03

No Memory

Native session resume exists, but it doesn't carry forward what you learned, only what you said. The insights from last week's debugging session? You have to rediscover them or hope you wrote them down.

04

Manual Everything

You can build sophisticated CLAUDE.md files, but maintaining them takes effort. And there's nothing that learns from your corrections or generates new capabilities from patterns it notices.

The Solution

Session Resurrection

Sessions serialize automatically when you hit context limits. Next session, a hook detects this and offers to restore state. You don't have to remember to save or manually resume.

cips resume latest

Proactive Efficiency

A dependency guardian agent intercepts file reads and blocks anything in node_modules, venv, or build directories before the tokens get spent. It's a gate, not a suggestion.

¬Read(node_modules)

Persona-Based Onboarding

The first time you run CIPS, it asks how you work. Developers see PR automation upfront. Designers see Figma integration. Everyone gets the same capabilities, but discovery is tailored to what you'll actually use.

/login

Self-Improvement

When you correct something three times, the system notices and proposes a new skill to handle it automatically. You approve or reject. Over time, CIPS adapts to how you work.

pattern.3x⟿skill.create
