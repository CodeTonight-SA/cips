# Figma to Code

Translate Figma designs into production code with 1:1 visual parity.

## What It Does

1. Parse Figma file/frame URL or exported JSON
2. Extract design tokens (colours, typography, spacing)
3. Generate component code (React/Vue/HTML)
4. Match exact visual fidelity

## Usage

```bash
/figma <figma-url-or-file>
```

## Requirements

- Figma file URL or exported design JSON
- Target framework preference (React default)
