# Reverse Engineer API

Systematically reverse engineer authenticated web APIs from browser DevTools captures.

## What It Does

1. Capture network requests from DevTools
2. Build authenticated API client
3. Map parameters and validation rules
4. Generate monitoring alerts for changes

## Usage

```bash
/reverse-api
```

## Requirements

- Browser DevTools HAR export or network log
- Your own credentials (legal requirement)
