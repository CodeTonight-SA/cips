# ui-complete

Invoke the unified UI development skill that combines:

1. **frontend-design** (plugin): Anti-AI-slop aesthetics, distinctive typography, bold visual direction
2. **mobile-responsive-ui** (CIPS): dvh units, 48px touch targets, mobile-first, testing protocol
3. **ENTER Konsult brand** (when applicable): Swiss Minimalism, refined elegance

## 4-Phase Workflow

1. **Aesthetic Direction**: Choose bold visual direction (not generic AI aesthetics)
2. **Mobile-First Implementation**: Start at 375px, use min-width queries
3. **Responsive Audit**: Test at 375px, 768px, 1920px
4. **Brand Application**: Apply project brand if configured

## Usage

```
/ui-complete              # General UI task
/ui-complete hero section # Specific component
```

## Requirements Enforced

- Distinctive typography (no Inter, Roboto, Arial)
- Cohesive color palette (no purple gradients)
- 48px touch targets (min-h-12 min-w-12)
- dvh fallbacks (h-screen h-dvh)
- Mobile-first classes (unprefixed = mobile)
